webpack简单总结

  • config/index.js dev环境
    • autoOpenBrowser: true 自动打开浏览器
    • useEslint: false 关闭eslint检查

  • config/index.js build环境

    • assetsRoot: path.resolve(__dirname, '../dist') 产品文件的存放路径
    • assetsSubDirectory 二级目录，存放静态资源文件的目录，位于dist文件夹下
    • assetsPublicPath 属性作用是指定编译发布的根目录,发布路径,如果构建后的产品文件有用于发布CDN或者放到他域名的服务器,可以在这里进行设置,设置之后构建的产品文件在注入到index.html中的时候就会带上这里的发布路径
    • productionSourceMap: false 减少打包文件大小
    • productionGzip: true 开启gzip压缩 要先安装npm install --save-dev compression-webpack-plugin

  • build/webpack.prod.conf

    • new HtmlWebpackPlugin({hash: true}) 使浏览器无法缓存,发布新版本
