js常用方法

发表于 2017-12-18 | 分类于 javascript

  • Array常用方法

    • arr.push() //向数组最后添加一个或多个元素,并返回长度
    • arr.unshift() //向数组最前面添加一个元素,并返回长度
    • arr.concat() // 连接数组
    • arr.join() //在数组元素中间加入分隔符
    • arr.slice(start,end) //截取从start开始到end之前,不包含end,返回新数组,不会改变旧数组
    • arr.splice(start,删除个数,增加的元素) //会改变元素,可以用来插入元素
    • arr.pop() // 删除数组最后一个元素并返回该元素,原数组改变
    • arr.shift() // 删除数组第一个元素并返回该元素,原数组改变
    • arr.indexOf(元素) // 可以获取元素在数组中的索引
    • arr.findIndex() //可以获取指定条件的数据所在的索引
    • arr.filter(callback(ele,index,array)) //返回一个布尔值,并且将返回值为true的元素装入一个新数组返回

  • string常用方法

    • str.indexOf(“str”,index) //检索字符串,如果有index将从指定位置开始
    • str.replace(regexp/str,replacement) //替换正则或者字符串,replacement要替换的str
    • str.slice(start,end) //截取从start开始的字符串,并返回一个新数组
    • str.split(regexp/str,howmany) //分割字符串
    • str.toLocaleLowerCase() //将字符串转为小写
    • str.toLocaleUpperCase() //将字符串转为大写
    • str..link(“url”) //将字符串显示为链接

  • Math常用方法

    • Math.abs() //返回绝对值
    • Math.ceil() //向上取整
    • Math.floor() //向下取整
    • Math.max(x,y) //返回最大值
    • Math.min(x,y) //返回最小值
    • Math.random() //返回0-1之间的随机数
    • Math.round() //四舍五入

  • string常用方法

    • indexOf() 检索字符串中是否存在这个字符串 是返回索引,否返回-1
    • replace(regexp,替换字符串) 替换与正则表达式匹配的子串

  • reg 正则对象.test(字符串)

  • DOM常用方法

    • document.createElement() //生成新标签
    • element.appendChild() //向元素添加子节点,作为最后一个子节点
    • element.getAttribute(属性名); //获取自定义属性,自定义属性无法点出来
    • element.setAttribute(属性名，属性值); //设置添加
    • element.removeAttribute(属性名); //删除
    • element.removeChild(child); //删除节点必须是父元素.出来
    • element.replaceChild(newChild,oldChild) //替换节点
    • element.className //设置类名
      • H5设置类名
      • .classList.add() //添加类名
      • .classList.remove() //删除类名
      • .classList.toggle() //开关类名
    • element.innerHTML //获取内容包括标签
    • element.value //获取表单元素的内容
    • element.children //获取所有子元素集合
    • element.parentNode //返回元素的父节点
    • element.firstElementChild //第一个子元素
    • element.lastElementChild //最后一个子元素
    • element.cloneNode(false/true) //是否深度克隆,true连带子节点
    • element.insertBefore(newChild,oldChild) //在old之前插入new
    • 方位
      • element.offsetWidth //返回元素宽度
      • element.offsetHeight //返回元素高度
      • element.offsetTop //返回元素宽度
      • element.offsetLeft //返回元素宽度
    • 滚动条
      • scrollLeft //往左滚动的距离
      • scrollTop // 往上滚动的距离
  • event对象常用方法
    • onabort 图像的加载被中断。
    • onblur 元素失去焦点。
    • onchange 域的内容被改变。
    • onclick 当用户点击某个对象时调用的事件句柄。
    • ondblclick 当用户双击某个对象时调用的事件句柄。
    • onerror 在加载文档或图像时发生错误。
    • onfocus 元素获得焦点。
    • onkeydown 某个键盘按键被按下。
    • onkeypress 某个键盘按键被按下并松开。
    • onkeyup 某个键盘按键被松开。
    • onload 一张页面或一幅图像完成加载。
    • onmousemove 鼠标被移动。
    • onmouseout 鼠标从某元素移开。
    • onmouseover 鼠标移到某元素之上。
    • onmousedown 鼠标按钮被按下。
    • onmouseup 鼠标按键被松开。
    • onreset 重置按钮被点击。
    • onresize 窗口或框架被重新调整大小。
    • onselect 文本被选中。
    • onsubmit 确认按钮被点击。
    • onunload 用户退出页面。
    • 属性
      • altKey 返回当事件被触发时，”ALT” 是否被按下。
      • shiftKey 返回当事件被触发时，”SHIFT” 键是否被按下。
      • ctrlKey 返回当事件被触发时，”CTRL” 键是否被按下。
      • metaKey 返回当事件被触发时，”meta” 键是否被按下。
      • button 返回当事件被触发时，哪个鼠标按钮被点击。
      • relatedTarget 返回与事件的目标节点相关的节点。
      • screenX 返回当某个事件被触发时，鼠标指针的水平坐标。 用户屏幕
      • screenY 返回当某个事件被触发时，鼠标指针的垂直坐标。
      • clientX 返回当事件被触发时，鼠标指针的水平坐标。浏览器当期窗口
      • clientY 返回当事件被触发时，鼠标指针的垂直坐标。
      • pageX 整个页面的水平坐标
      • pageY 整个页面的垂直坐标

  • window常用方法

    • window.open(url) //打开一个连接
    • window.close() //关闭窗口
    • window.onload=function //在页面加载完成执行
    • window.getComputedStyle(element,null)[key] //不加key获取一个元素的所有属性,加key是获取指定属性的value

  • 注册事件的三种方法

    • 事件源+(on+事件类型)=function(){事件执行} //只能绑定一次，如果多次绑定，会覆盖之前的绑定
      • 阻止冒泡方法：
        • 1 event.stopPropagation() 在当前元素的某个事件处理程序里阻止
        • 2 event.cancelBubble = true; ie8
    • 事件源+addEventListener(事件类型+function(){事件执行},false) //false阻止冒泡,可以绑定多次
    • 事件源.attachEvent(“on”+事件类型,事件处理程序); //ie6-10

  • JQuery

    • 多库共存 jQuery.noConflict() 切断与$的联系
    • $(function(){ code }) //接口函数
    • dom对象转jq对象 $(dom对象) || jq对象转换为dom对象 jq对象[索引]
    • $(CSS选择器) //jq对象
      • $(选择器:even) 偶数选择器
      • $(选择器:odd) 奇数选择器
    • 操作class
      • jq.addClass(类名) 添加类名
      • jq.removeClass(类名) 删除类名
      • jq.toggleClass(类名) 有就删除,没有就添加
    • 操作css
      • jq.css(属性名); 获取属性名,下面都是设置
      • jq.css(属性名,属性值);
      • jq.css(属性名和属性值的键值对集合);
      • jq.css(属性名，属性值).css(属性名，属性值);
    • 管理属性
      • jq对象.attr(属性名); – 获取
      • jq对象.attr(属性名,属性值); – 设置
      • jq对象.prop(属性名); – 获取
      • jq对象.prop(属性名,属性值); – 设置
      • jq对象.width(值); 设置宽
      • jq对象.height(值); 设置高
    • 获取元素方位
      • jq对象.offset(键值对集合); 获取设置偏移量
      • jq对象.position(); – 只能得到，不能设置
    • 获取设置滚动距离
      • jq对象.scrollTop(值);
      • jq对象.scrollLeft(值)
    • 获取元素内容
      • jq对象.val(内容); – 设置表单元素
      • jq对象.html(内容); – 设置普通元素
      • jq对象.text(内容); – 设置普通元素
    • 注册事件
      • jq对象.事件名称(回调函数);
      • jq对象.on(事件类型,(子代元素选择器),处理程序);
    • 移除注册事件
      • jq对象.off(事件类型)
    • jq对象.index()方法 获取的是某个元素在其兄弟元素之间的索引
    • jq对象.eq(index) 方法，获取的是jq对象中索引为index的dom对象的jq对象
    • jq对象.userinfo(‘内容’) 可以查找文字内容对应的对象
    • jq遍历方法
      • jq对象.each(function(index,element){}
      • element 就是指遍历到的某个元素对象
      • index 当前DOM对象的索引
    • 操作节点
      • $(“<标签名>”);$(“<标签名></标签名>”);
      • parent.append(child); || child.appendTo(parent); 添加元素为最后一个子元素
      • parent.prepend(child)||child.prependTo(parent) 追加元素作为第一个子元素
      • jq对象.after(jq对象);||jq对象.before(jq对象); 插入元素
      • jq对象.remove(); 删除节点
      • jq对象.empty(); 清空节点
      • jq对象.clone(true|false); 克隆节点 true 复制事件 false 不复制事件
      • jq对象.parent(筛选选的选择器); 找到父元素
      • jq对象.parents(筛选选的选择器); 找到所有父辈祖父辈
      • jq对象.children(筛选选的选择器); 找到子元素
      • jq对象.find(筛选选的选择器); 找到所有后代元素
      • jq对象.siblings(筛选条件); 找到兄弟元素
      • jq对象.next(也可以筛选); 下一个兄弟元素
      • jq对象.nextAll(可以筛选); 接下来全部兄弟元素
    • 操作动画
      • jq对象.animate(属性,速度,回调); 动画函数
      • jq对象.hide(speed,callback); 隐藏动画
      • jq对象.show(speed,callback); 显示动画
      • jq对象.toggle(speed,callback); 隐藏就显示,显示就隐藏
      • jq对象.fadeOut(speed,callback); 淡出
      • jq对象.fadeIn(speed,callback); 淡入
      • jq对象.fadeToggle(speed,callback); 淡出就淡入,淡入就淡出
      • jq对象.fadeTo(speed,opacity,callback); 淡入到哪个透明度
      • jq对象.slideUp(speed,callback); 滑动收起
      • jq对象.slideDown(speed,callback); 滑动放下来
      • jq对象.slideToggle(speed,callback); 收起就放下,放下就收起
      • jq对象.stop(stopAll,goToEnd);放在动画前面 stopAll 停止所有一般为true gotoend： 是否到达结尾 一般为false (默认)
    • jq对象.hover(鼠标移入的函数，鼠标移出的函数);

  • H5+C3(使用3D属性必须开启transform-style:preserve-3d; )

    • box-sizing:border-box 怪异模式
    • text-shadow:水平位移 垂直位移 模糊值 颜色 //文本阴影
    • box-shadow:水平位移 垂直位移 模糊值 外延值 颜色 //盒子阴影
    • border-radius //圆角
    • 增大响应区域
      • background-origin: content-box; background-clip: content-box; padding;
    • linear(线)-gradient(渐变)参数: 方向 ,颜色 开始的位置 ,颜色 结束的位置
    • radial-gradient 径向渐变 参数: (形状 椭圆 ellipse| 正圆 circle 半径的大小 原点位置 , 颜色 开始的位置 ,颜色结束位置 )
    • transition: property duration function-timing delay 过渡效果
    • transform: translate(x,y) 百分比相对自身 translate3d(x,y,z)
    • transform: rotate(0deg); 旋转顺时针 rotate3d(x,y,z)
    • transform: scale(width,height) 缩放,倍数
    • animation:name duration timing delay iteration-count 动画函数 @keyframes name{0%/form{} 100%/to{})
    • backface-visibility:hidden; 反面不可见
