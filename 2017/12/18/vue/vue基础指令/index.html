基础指令(当做属性用在标签中)

  • v-cloak 配合[v-clock]{display:none}解决vue还没有加载完成的闪烁问题
  • v-text 等于innertext
  • v-html 等于innerHTML
  • v-model 双向数据绑定
  • v-bind 可以实现非v-model赋予的值,进行双向数据绑定
    • v-bind:value=’key’
    • 可以简写为一个:
    • 特殊用法　<a v-bind="{href:'del/msg='+msg}"> //将字段变成对象,属性还是属性,值变成常量+变量
  • v-for的三种用法,必加:key=”index”,这样不会造成整个数据重构,只会改变当前值数据
    • <li v-for="value in list" :key="index"></li> 遍历数组
    • <li v-for="(value,index) in list" :key="index">--</li> 遍历数组
    • <li v-for="(value,key,index) in user" :key="index">----</li> 遍历对象
  • v-on://触发事件,简写v-on: 可以换成@, 不传参可以不加(),传参才加()
    • <p v-on:click="getvalue"></p>
    • <input type="text" @keydown="run($event)">
    • <input type="text" @keydown.13="run"> //按键修饰符
    • @keydown.enter="keyD"
  • v-if和v-show,用true和false控制显示和隐藏
    • <p v-if="Boolean"></p> //直接删除dom元素,造成重构
    • <p v-show="Boolean"></p> //将dom元素用display:none隐藏起来,性能更好
  • ref=name
    • this.$refs.name.focus() 效果相当于doc.elbyid().focus(),建议用这种方式操作dom元素
  • prevent修饰符,阻止浏览器默认事件
    • @submit.prevent="submit"
  • stop修饰符可以阻止冒泡，防止触发无关的父元素事件
